# About
TOL - Text Organization Language

# Syntax
```
>Header
^Big text
* small text
default text
;;;(void)
🔴Red
🟢Green
🔵Blue
🟡Yellow
[image.png]
{yoururl.com}
```
# Rules
* file must be a .tol
