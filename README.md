# About
TOL - Text Organization Language

# Syntax
```
>Header
^Big text
* small text
default text
;;;(void)
游댮Red
游릭Green
游댯Blue
游리Yellow
[image.png]
{yoururl.com}
```
# Rules
* file must be a .tol
